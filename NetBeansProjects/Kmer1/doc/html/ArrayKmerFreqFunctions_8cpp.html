int &  nElements,
std::string  validNucleotides 
)

Normalizes the given array of KmerFreq. That is, for each Kmer in the array, all its characters are converted to uppercase. Then, invalid characters are replaced by the UNKNOWN_NUCLEOTIDE value.

If after the previous normalization process of every kmer, identical kmers are obtained, these will be merged into the first identical kmer by adding their frequencies. For example, suposse the following list of kmers: 4 Ct 5 hG 4 nG 1 cT 4

After the process of normalization of every kmer, we obtain the following list of kmers:

4 CT 5 _G 4 _G 1 CT 4

The final step will transform the list into: 2 CT 9 _G 5

Parameters
arrayAn array of KmerFreq. Input/output parameter
nElementsThe number of elements used by the array. Output parameter
validNucleotidesa string with the list of characters (nucleotides) that should be considered as valid. Input parameter
